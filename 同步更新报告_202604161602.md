## 配置说明

百炼图片生成需要在 `ecommerce_config.json` 中配置API密钥：

```json
{
  "bailian": {
    "api_key": "sk-xxxxxxxx",
    "region": "cn-beijing",
    "enabled": true
  }
}
```

详细配置请参考: `百炼图片生成对接指南.md`
