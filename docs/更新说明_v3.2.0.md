# SellAI v3.2.0 更新说明

## 发布信息
- **版本**: v3.2.0
- **发布日期**: 2025-04-11

本次更新从长期任务源码目录整合了95个核心模块到SellAI部署包，实现了290个Python文件的完整整合。

## 迁移说明
- 所有新增模块使用try-except导入模式，确保兼容性
- 未安装的模块会记录warning日志但不会中断启动
- 保持了原有v3.1.0及之前版本的所有功能不变

## 已知限制
- 部分模块可能需要额外的依赖包
- 语音服务模块需要系统安装对应的音频库
- 网络服务器模块需要配置网络权限
