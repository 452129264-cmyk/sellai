## 核心新增模块

### 1. 自我进化大脑 (Self-Evolution Brain) ⭐最高优先级

#### 模块概述
自主迭代进化系统，实现全自动化自主进化，无需人工干预即可持续优化系统表现。

#### 核心功能
1. **每日复盘引擎** (`/api/v2/evolution/review`)
   - 自动收集并分析系统运行数据
   - 识别成功模式和失败案例
   - 生成优化建议
   - 与Notebook LM知识底座深度集成

2. **策略优化器** (`/api/v2/evolution/optimize`)
   - 基于复盘结果自动优化策略
   - 支持成本优化、转化率优化、资源分配优化
   - A/B测试框架集成
   - 实时性能监控与反馈

3. **经验沉淀管理器** (`/api/v2/evolution/experience`)
   - 提取并固化成功模式
   - 分析失败案例并生成警示
   - 构建经验知识库
   - 跨分身经验共享

4. **配置管理器**
   - 灵活的策略配置
   - 自定义复盘时间表
   - 优化目标设定

#### 新增API端点
```
POST /api/v2/evolution/review     - 每日复盘
POST /api/v2/evolution/optimize  - 策略优化
POST /api/v2/evolution/experience - 经验沉淀
GET  /api/v2/evolution/status     - 进化状态
```

#### 文件结构
```
src/self_evolution_brain/
├── __init__.py
├── main_controller.py      # 主控制器
├── daily_review_engine.py  # 每日复盘引擎
├── strategy_optimizer.py   # 策略优化器
├── experience_persistence.py # 经验沉淀
└── config_manager.py       # 配置管理
```

### 2. Memory V2 分层记忆系统

#### 模块概述
验证通过后才构建索引的记忆系统，支持热/温/冷分层存储策略。

#### 核心功能
1. **记忆验证器** (`/api/v2/memory/validate`)
   - 验证记忆数据完整性
   - 检查数据一致性
   - 质量评分
   - 支持标准、严格、自定义验证级别

2. **分层索引器** (`/api/v2/memory/index`)
   - 为验证通过的记忆创建索引
   - 支持多种索引类型（全文、关键词、语义、关系）
   - 分层索引策略（热/温/冷）
   - 异步索引构建

3. **集成管理器** (`/api/v2/memory/manage`)
   - 创建、更新、删除记忆
   - 查询记忆（支持模糊搜索、语义搜索）
   - Coze记忆与本地验证记忆的同步
   - 查询缓存加速

#### 新增API端点
```
POST /api/v2/memory/validate - 记忆验证
POST /api/v2/memory/index    - 创建索引
POST /api/v2/memory/manage   - 记忆管理（增强）
GET  /api/v2/memory/status   - 记忆状态
```

#### 文件结构
```
src/memory_v2/
├── __init__.py
├── memory_v2_validator.py    # 验证器
├── memory_v2_indexer.py      # 索引器
└── memory_v2_integration.py  # 集成管理器
```
