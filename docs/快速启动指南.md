# SellAI v3.0.0 快速启动指南
## 终极完整版

---

## 环境要求

- Python 3.10+
- PostgreSQL 14+ (可选)
- Redis 6+ (可选)

---

## 安装步骤

### 1. 克隆或下载项目

```bash
git clone <repo_url>
cd SellAI_v3.0.0
```

### 2. 创建虚拟环境

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

### 4. 配置环境变量 (可选)

```bash
export DATABASE_URL="sqlite:///./data/sellai.db"
export REDIS_URL="redis://localhost:6379"
```

### 5. 启动服务

```bash
python main.py
```

服务将在 `http://localhost:8000` 启动。

---

## Docker部署

### 构建镜像

```bash
docker build -t sellai:v3.0.0 .
```

### 运行容器

```bash
docker run -d \
    -p 8000:8000 \
    -v ./data:/app/data \
    -v ./logs:/app/logs \
    --name sellai \
    sellai:v3.0.0
```

## Railway部署

### 1. 连接GitHub仓库

在Railway控制台中连接您的GitHub仓库。

### 2. 配置环境变量

```env
DATABASE_URL=<your_postgres_url>
REDIS_URL=<your_redis_url>
PORT=8000
```

### 3. 设置启动命令

```bash
pip install -r requirements.txt && python main.py
```

### 4. 部署

点击"Deploy"按钮开始部署。

## 故障排除

### 问题1: 端口被占用

```bash
# 查找占用端口的进程
lsof -i :8000

# 或使用其他端口
python main.py --port 8080
```

### 问题2: 模块导入失败

检查是否所有依赖已正确安装：
```bash
pip install -r requirements.txt
```

### 问题3: 数据库连接失败

确保数据库服务正在运行，或使用SQLite作为默认数据库。

### 问题4: 权限错误

确保日志目录可写：
```bash
chmod 755 logs
chmod 755 data
```
