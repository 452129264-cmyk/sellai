## 三、关键修复点

### threshold参数修复
```python
# ❌ 旧代码 (Railway当前运行)
threshold: float = Query(60.0, ...)  # 使用Query，JSON body不生效

# ✅ 新代码 (本地已验证正确)
threshold: float = Body(45.0, ...)  # 使用Body，JSON body正确传递
```

### 毛利门槛调整
- 原门槛: 60% (过高，连续多日无60%+商机)
- 新门槛: 45% (匹配数据源实际毛利分布30%-50%)
