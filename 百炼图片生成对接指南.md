## 功能概述

百炼图片生成适配器支持以下能力：

| 功能 | 模型 | 说明 |
|------|------|------|
| 文生图 | `wanx-v1` | 根据文字描述生成图片 |
| 图生图 | `image-synthesis` | 参考图片生成新图 |
| 虚拟模特 | `virtualmodel-v2` | 人台图生成真人模特 |
| AI试衣 | `ai-tryon` | 模特换装试穿 |
| 创意海报 | `wanx-poster` | 自动生成营销海报 |
| 背景生成 | `background-generation` | 产品图换背景 |

### 支持的风格

- `photography`: 摄影风格
- `portrait`: 人像风格
- `3d-catoon`: 3D卡通
- `anime`: 动画风格
- `oil_painting`: 油画风格
- `watercolor`: 水彩风格
- `sketch`: 素描风格
- `chinese_painting`: 中国画
- `minimalist`: 极简风格
- `commercial`: 商业摄影

## API密钥申请

### 步骤1：注册阿里云账号

1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 使用阿里云账号登录（如没有，需先注册）

### 步骤2：开通服务

1. 进入「模型服务」→「百炼服务」
2. 点击「开通服务」
3. 选择需要的模型服务（文生图、虚拟模特等）
4. 确认开通

### 步骤3：创建API Key

1. 进入「API-KEY管理」
2. 点击「创建API Key」
3. 复制生成的API Key（格式：`sk-xxxxxxxx`）

### 步骤4：充值（如需要）

百炼提供一定的免费额度，具体请查看官方定价：
- 百炼平台：https://bailian.console.aliyun.com/#/home

## 配置说明

### 配置文件位置

编辑 `ecommerce_config.json`，在 `bailian` 节点下配置：

```json
{
  "bailian": {
    "api_key": "sk-your-api-key-here",
    "base_url": "https://dashscope.aliyuncs.com",
    "models": {
      "text2image": "wanx-v1",
      "image2image": "image-synthesis",
      "virtual_model": "virtualmodel-v2",
      "ai_tryon": "ai-tryon",
      "poster": "wanx-poster",
      "background": "background-generation"
    },
    "timeouts": {
      "task_submit": 30,
      "task_query": 10,
      "task_poll": 60
    },
    "polling": {
      "max_attempts": 60,
      "interval": 2
    }
  }
}
```

### 配置项说明

| 配置项 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| `api_key` | string | 百炼API密钥 | 空 |
| `base_url` | string | API基础URL | https://dashscope.aliyuncs.com |
| `models.*` | string | 各功能对应的模型名称 | 见默认值 |
| `timeouts.task_submit` | int | 任务提交超时（秒） | 30 |
| `timeouts.task_query` | int | 任务查询超时（秒） | 10 |
| `polling.max_attempts` | int | 最大轮询次数 | 60 |
| `polling.interval` | int | 轮询间隔（秒） | 2 |

## API参考

### BailianImageAdapter

主适配器类，提供所有图片生成功能。

#### 初始化

```python
adapter = BailianImageAdapter(config_path="ecommerce_config.json")
```

#### 方法

##### generate_text2image()

文生图 - 根据文字描述生成图片

```python
result = await adapter.generate_text2image(
    prompt: str,                    # 必填，图片描述
    style: str = None,              # 可选，风格
    negative_prompt: str = None,    # 可选，负面提示词
    size: str = "1024*1024",       # 可选，图片尺寸
    n: int = 1,                     # 可选，生成数量
    seed: int = None,              # 可选，随机种子
    wait: bool = True,              # 可选，是否等待完成
    timeout: int = 120              # 可选，超时时间（秒）
)
```

##### generate_image2image()

图生图 - 参考图片生成新图

```python
result = await adapter.generate_image2image(
    prompt: str,                    # 必填，图片描述
    ref_image_url: str,             # 必填，参考图片URL
    style: str = None,              # 可选，风格
    strength: float = 0.7,          # 可选，生成强度
    size: str = "1024*1024",        # 可选，图片尺寸
    n: int = 1,                     # 可选，生成数量
    wait: bool = True,
    timeout: int = 120
)
```

##### generate_virtual_model()

虚拟模特 - 人台图生成真人模特

```python
result = await adapter.generate_virtual_model(
    product_image_url: str,          # 必填，产品/人台图URL
    model_type: str = "female_casual", # 可选，模特类型
    size: str = "1024*1024",
    wait: bool = True,
    timeout: int = 180
)
```

##### generate_ai_tryon()

AI试衣 - 模特换装

```python
result = await adapter.generate_ai_tryon(
    garment_image_url: str,          # 必填，服装图片URL
    model_image_url: str,            # 必填，模特图片URL
    size: str = "1024*1024",
    wait: bool = True,
    timeout: int = 180
)
```

##### generate_poster()

创意海报生成

```python
result = await adapter.generate_poster(
    prompt: str,                     # 必填，海报描述
    size: str = "1024*1782",
    wait: bool = True,
    timeout: int = 120
)
```

##### generate_background()

图像背景生成

```python
result = await adapter.generate_background(
    product_image_url: str,          # 必填，产品图片URL
    background_prompt: str,           # 必填，背景描述
    size: str = "1024*1024",
    wait: bool = True,
    timeout: int = 120
)
```

#### 返回值

`BailianImageResult` 对象：

```python
@dataclass
class BailianImageResult:
    request_id: str          # 请求ID
    success: bool            # 是否成功
    task_id: Optional[str]   # 任务ID
    images: List[Dict]      # 图片列表 [{url: str}, {base64: str}]
    status: Optional[str]    # 状态
    metadata: Dict           # 元数据
    error_message: Optional[str]  # 错误信息
    processing_time: Optional[float]  # 处理时间（秒）
```

## 代码示例

### 示例1：基础文生图

```python
import asyncio
from src.bailian_image import BailianImageAdapter, BailianImageStyle

async def main():
    adapter = BailianImageAdapter()
    
    result = await adapter.generate_text2image(
        prompt="专业电商产品摄影，女士手提包，皮革质感，",
              "白色背景，高清细节，光线均匀，商业摄影风格",
        style=BailianImageStyle.PHOTOGRAPHY,
        size="1024*1024"
    )
    
    print(f"成功: {result.success}")
    if result.images:
        print(f"图片URL: {result.images[0]['url']}")
    
    await adapter.close()

asyncio.run(main())
```

### 示例5：批量生成与轮询

```python
async def batch_generate():
    adapter = BailianImageAdapter()
    
    prompts = [
        "白色运动鞋，侧面角度",
        "白色运动鞋，正面角度",
        "白色运动鞋，背面角度"
    ]
    
    tasks = []
    for prompt in prompts:
        result = await adapter.generate_text2image(
            prompt=prompt,
            wait=False  # 不等待，快速提交
        )
        tasks.append(result.task_id)
    
    # 批量轮询
    for task_id in tasks:
        task = await adapter.wait_for_completion(task_id)
        print(f"Task {task_id}: {task.status.value}")
    
    await adapter.close()
```

## 常见问题

### Q1: API调用返回401错误

**原因**：API密钥未配置或已过期

**解决方案**：
1. 检查 `ecommerce_config.json` 中的 `bailian.api_key` 是否正确
2. 确认API密钥在阿里云百炼平台有效
3. 检查是否已开通对应服务的额度

### Q2: 任务提交成功但查询不到

**原因**：任务可能还在队列中或服务暂时不可用

**解决方案**：
1. 等待几秒后重试查询
2. 检查百炼平台的服务状态
3. 确认模型名称是否正确

### Q3: 任务一直处于RUNNING状态

**原因**：
- 服务繁忙，生成时间较长
- 网络问题导致轮询失败

**解决方案**：
1. 增加轮询超时时间
2. 检查网络连接
3. 查看百炼平台是否有服务异常通知

### Q4: 返回的图片URL无法访问

**原因**：
- URL已过期（百炼图片URL有效期有限）
- 图片仍在生成中

**解决方案**：
1. 及时下载图片到本地存储
2. 使用 `base64` 格式获取图片数据
3. 确认任务状态为 `SUCCEEDED`

### Q5: 如何处理高并发场景？

**建议**：
1. 使用连接池管理HTTP客户端
2. 限制并发任务数量
3. 考虑使用异步批量提交

```python
# 推荐：使用信号量控制并发
semaphore = asyncio.Semaphore(5)

async def limited_generate(prompt):
    async with semaphore:
        return await adapter.generate_text2image(prompt)
```

### Q6: 图片尺寸有哪些选择？

**支持的尺寸**：
- `512*512`: 正方形小图
- `768*768`: 正方形中图
- `1024*1024`: 正方形大图（推荐）
- `1024*512`: 横版横幅
- `1024*1782`: 竖版海报
- `1024*2048`: 竖版长图

### Q7: 如何获取图片的base64数据？

```python
result = await adapter.generate_text2image(
    prompt="产品图",
    output_type="base64"  # 请求返回base64
)

if result.images and 'base64' in result.images[0]:
    img_data = result.images[0]['base64']
    # 可以直接用于前端显示或保存
```
