# SellAI v3.2.3 部署指南

## 更新内容

### 1. API配置模块（新增）
- `src/api_config/` - 统一API配置管理
  - `config.py` - DeepSeek + 百炼配置
  - `deepseek_client.py` - DeepSeek推理引擎
  - `bailian_client.py` - 百炼图片生成

### 2. 新增API端点
- `GET /api/config/api-status` - 查看API配置状态
- `POST /api/config/test-deepseek` - 测试DeepSeek连接
- `POST /api/config/test-bailian` - 测试百炼连接
- `POST /api/image/generate` - 图片生成
- `POST /api/image/product` - 电商产品图生成

### 3. 预测性记忆增强
- 预测引擎集成DeepSeek API
- LLM增强的预测建议

## 部署步骤

### 1. 获取API Key

**DeepSeek（必填）**:
1. 访问 https://platform.deepseek.com/
2. 注册并创建 API Key
3. 充值最低 ¥10（约等于免费）

**百炼（可选）**:
1. 访问 https://bailian.console.aliyun.com/
2. 开通百炼服务
3. 创建 API Key

### 2. Railway配置环境变量

在 Railway 项目设置中添加：
```
DEEPSEEK_API_KEY=sk-你的deepseek_key
BAILIAN_API_KEY=你的百炼key
```

### 3. 推送代码

```bash
cd SellAI部署包
git init
git remote add origin https://github.com/452129264-cmyk/sellai.git
git add -A
git commit -m "feat: v3.2.3 集成DeepSeek+百炼API"
git push -u origin main --force
```

### 4. 验证部署

等待 Railway 部署完成后：

```bash
# 检查API状态
curl https://你的域名/api/config/api-status

# 测试DeepSeek
curl -X POST https://你的域名/api/config/test-deepseek

# 测试预测性记忆
curl -X POST https://你的域名/api/memory/predict \
  -H "Content-Type: application/json" \
  -d '{"context": {"event": "新产品上架"}, "prediction_type": "comprehensive"}'
```

## 故障排除

### 502错误
- 健康检查已优化到120秒
- 如果仍有问题，检查Railway日志

### API连接失败
- 检查环境变量是否正确设置
- 确认API Key未过期
- 查看 `/api/config/api-status` 确认配置

### 预测功能不可用
- 确认 DeepSeek API Key 已配置
- 检查 DeepSeek 账户余额
