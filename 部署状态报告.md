## 🚀 快速部署指南

### 1. 安装依赖
```bash
cd SellAI部署包
pip install -r requirements_complete.txt
```

### 2. 运行模块检查
```bash
python check_modules.py
```

### 3. 启动服务
```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 4. 访问API文档
- Swagger: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc
