## 🚀 快速部署

### 方式一：一键部署（推荐）

```bash
cd SellAI部署包
./deploy.sh
```

### 方式二：手动部署

```bash
# 1. 安装依赖
cd SellAI部署包
pip install -r requirements_complete.txt

# 2. 检查模块
python3 check_modules.py

# 3. 启动服务
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 方式三：Docker部署

```bash
cd SellAI部署包
docker build -t sellai:v3.2.0 .
docker run -p 8000:8000 sellai:v3.2.0
```

## 🔍 故障排查

### 问题：模块导入失败
```bash
# 重新安装依赖
pip install -r requirements_complete.txt --force-reinstall

# 检查模块状态
python3 check_modules.py
```

### 问题：端口被占用
```bash
# 使用其他端口
uvicorn main:app --host 0.0.0.0 --port 8080
```

### 问题：数据库错误
```bash
# 清理并重建数据库
rm -rf data/
mkdir -p data/shared_state
python3 -c "from main import app_state"
```
